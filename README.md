# Cognifyz Technologies Full Stack Development Internship Tasks

## Internship Duration
October 5, 2023 - November 5, 2023

## Overview
This repository contains all the tasks completed during my Full Stack Development Internship at Cognifyz Technologies. The projects demonstrate progressive learning and implementation of various web development technologies and concepts.

## Project Structure

The repository is organized into multiple task folders, each building upon previous concepts:

### Basic Server Setup (Task01-02)
- Basic Express.js server configuration
- Template engine integration
- Static file serving
- View rendering

### Frontend Integration (Task03-04)
- Public assets management
- Enhanced view templates
- Static file handling
- Responsive web design

### Data Management (Task05)
- Database integration
- Data models
- CRUD operations
- File system operations

### Advanced Features (Task06)
- Environment configuration
- Middleware implementation
- MVC architecture
- RESTful API routes
- MongoDB integration
- User authentication

### Additional Features (Task07-08)
- Enhanced security features
- Advanced database operations
- File upload functionality
- Session management

## Technologies Used
- Node.js
- Express.js
- MongoDB
- HTML5/CSS3
- JavaScript
- Template Engines
- Environmental Configuration

## Getting Started

1. Navigate to any task directory:
```bash
cd Task0X
```

2. Install dependencies:
```bash
npm install
```

3. Run the server:
```bash
npm start
```

## Environment Setup
Some tasks require environment variables. Check `.env.example` files in respective directories for required configurations.

## Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)
- MongoDB (for database-related tasks)
